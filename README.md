# Cajero Automático

Simulador de cajero automático desarrollado en Python, con persistencia de datos en JSON.

## Funcionalidades

- Validación de PIN con bloqueo por intentos fallidos
- Consultar saldo
- Depositar dinero
- Retirar dinero (con límite diario)
- Cambiar PIN
- Ver historial de movimientos
- Donaciones a fundación infantil

## Tecnologías

- Python
- JSON (persistencia de datos)

## Cómo ejecutarlo

```bash
python main.py
```

## Aprendizajes

Este proyecto me permitió practicar validación de datos, manejo de archivos JSON, control de flujo con menús interactivos, y estructuración de lógica de negocio en Python.
